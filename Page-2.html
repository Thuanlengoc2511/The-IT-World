Project Ideas

Overview

The AI project is beneficial for humans in many fields of work. Nowadays, the job demand is on the rise, so I find it difficult to controller a company with only human forces. To deal with this issue, I create this bot – Tori – to controller the company base on the CEO command. Tori can help a company set up the plan, keep the program on track. It also can work in many industries and security. This is a multi-function AI. There will be two version of Tori. One is small Tori for everyone to use, one is big Tori for a big company or a school. . For example, Tori can controller the student's study plan and give more exercises to support the real model. Tori can send the notification to the teacher how well did the student going on. These are only a few samples. There will be more in the future.

Motivation

My dream is to become an AI engineer. I want to create an AI which can think by itself just like a human. This project is also interesting to me because it is nearly an impossible challenge with our technologies. The feeling of making an AI that can think, learn, and follow human command makes me excited. An AI which I want to make is JARVIS in Iron Man movies. However, today technology cannot create an AI like that yet, and I have not had enough knowledge about AI technology. So I decided to make a simple AI, I will make a system controller AI call Tori.

Learning

‘Learning’ this is the best thing Tori can do. It can learn from experience from time to time, from working with humans. It will save the data in its memory and replay the data if Tori meet the same situation. Moreover, Tori not only learn from human but also learns from the internet. If Tori is not in the rush hour, it can go to the internet and learn how people work in various situations. This will help Tori get more information and experience to work more efficiently. Furthermore, this will make Tori more user-friendly and feel like a human

Analyst

Tori analyst plans for a company or an enterprise to achieve the best result. For example, in R&D (Research and development), Tori can base on the company's or enterprise's information to build more optimal processes to make the best product with the lowest prices. Tori is a high-tech AI, so it has a fantastic analyst ability. Moreover, Tori can be used in Education. Tori can analyst the strength and the weakness of the course and give solutions to improve it. 

Detecting

‘Detecting’ is the third feature of Tori. Detect mean Tori will find and fix or tell the manager to solve it. This ability can be used in Education as well. Tori can track student's study process and notify the teacher if there are any problems. Or the students can use Tori to find learning resources and the best course for themself. Besides, this function can also apply to the transport industry. Tori can connect all of the cars together through wifi and indicates the best transport route. 

Controlling

‘Controlling’ Tori can control most of the technology device, which is allowed. For example, controlling the plane to go to dangerous places for research or rescue. Or guide a whole company's system to homogenize the system and increase work productivity and smoothly. Furthermore, Tori can control traffic to prevent traffic jams and accidents. 

Security

‘Security’ is Tory's last feature up to now. It can use this ability in the security industry, like security cameras. It can make actionable context and insights when presented with new or unknown information, making logical, inferred conclusions based on potential incomplete subsets of data, create solutions to a known problem to empower security teams to select the best path towards remediation. Combine with all of Tori's features, it can stop the threat immediately. With all of those features, Tori can become the best AI on over the world and help a whole country safer and work smoother than before.

Skills Required

To work with this AI project, it required you need to have some relevant skills.

Programming Languages

First is some fundamental expertise in programming languages; in this project, we will use Python and C++. 

Solid mathematical and algorithms knowledge

You can not become a good programmer without good at math; that why you should have solid mathematical and algorithms knowledge to work as an AI developer. 

Master different UNIX tools

To work in an AI job, the AI developer must master different UNIX tools such as awk, grep, cut, sort, find. So good command over UNIX Tools is required.

Efficiency in distributed computing skill 

AI jobs required us to work with extensive data and information that we can not use on a single machine. Having efficiency in distributed computing skill will help you work with many devices at the same time. 

Curious and creative mindset

This is very important in making new ideas, creative solutions, and functions in AI work. Those skills above are required to work in this AI project.
