Git & Github

What is Git & Github

1.Git

Git is a version control system which allow you to keep track on your computer programs and files and the changes that are made to them over time. Moreover, it also allow you to work with your co-workers on a programing, code.

2.Github

Github is a website that host Git server program to hold and store your project. It has several features such as bug tracking, feature requests, task management, continuous integration for every project.

How to use Github?

Basic Git commands

Step 1: You need to tell your computer who you are. This command below will help you do it.

-Git congif --global user.name "name" (to tell your name).

-Git config --global user.email "email" (to tell your email).

Step 2: You need to clone your repository into your computer.

-Git clone "put your repository's link in here" (to clone your repository).

Step 3: If you want to store your project on Github, you will need to do three commands below.

-Git add . (to add your project)

-Git commit -m "Message" (to note what are you doing in this project)

-Git push (to push your project into github)

Conflicts

If you meet this issue, you will need to follow these steps:

Step 1: You need to pull that project into your computer.

- Use git pull to pull the project into your computer.

Step 2: Work on the project that you just pull into your computer.

- After finish your work, you must use git add ., git commit, git push to store it on Github again.

Branching

Work with branch

Step 1: Prepare.

- Create an empty repository in your computer, create another file in that repository and commit that file.

Step 2: Create branch.

- Use command Git branch "Branchname", after that it will appear your current branch with "*" in the head.

Step 3: Change Branch.

- To add commit into the new branch use Git checkout "Branchname", then use add ., and commit new file.
